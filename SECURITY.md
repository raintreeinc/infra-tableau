# Security Policy

## Supported Versions

Currently supported versions of code are as follows:

| Version | Supported          |
| ------- | ------------------ |
| 1.0.0   | :white_check_mark: |
| 5.0.x   | :x:                |
| 4.0.x   | :white_check_mark: |
| < 4.0   | :x:                |
